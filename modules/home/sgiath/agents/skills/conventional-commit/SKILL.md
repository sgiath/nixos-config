# Conventional commit

The commit message must follow the [conventional commit specification](./reference/specs.md)

When the work is related to a Shortcut ticket, you must include it in this format: `Fixes sc-78039` - the `sc-<id>` is the format that Shortcut integration recognizes and links the commit automatically to the ticket. If there is no Shortcut ticket mentioned anywhere, you don't have to include it.
